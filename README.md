
# Calculator
A simple portfolio project

## Tech Stack

**Client:** React

**Server:** Spring Boot

## Deployment

To deploy frontend project

```bash
cd ./frontend
```


```bash
npm start
```
To deploy backend project

```bash
cd ./backend
```
```bash
./gradlew build
```
